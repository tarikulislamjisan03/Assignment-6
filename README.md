# 🛠️ DigiTools Platform

A modern digital tools marketplace where users can browse, and purchase premium digital products like AI writing tools, design templates, stock assets, and more.

## 🚀 Technologies Used

- React.js
- Tailwind CSS
- JavaScript
- React Toastify
- Lucide React
- React Icons

## ✨ Features

- 🛒 **Cart System** — Add products to cart, remove items, and proceed to checkout with real-time total calculation.
- 🔍 **Product Filtering** — Browse through curated digital tools with tag-based categorization like Best Seller, Popular, and New.
- 🔔 **Toast Notifications** — Get instant feedback with beautiful toast messages when adding or removing items from cart.
